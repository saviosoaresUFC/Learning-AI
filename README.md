# Learning AI 🤖🧠

Bem-vindo ao repositório **Learning AI**! Este projeto é dedicado aos estudos, práticas e experimentações na área de Inteligência Artificial, Machine Learning e Ciência de Dados.

## 📁 Sobre o Projeto

Este repositório atua como um portfólio de aprendizado contínuo, contendo diversos subprojetos independentes que exploram diferentes conceitos, algoritmos de IA e análises de dados. 

Alguns dos projetos contidos aqui incluem:
- **`preco_apto/`**: Modelagem e previsão de preços de apartamentos.
- **`social_media_advertising/`**: Análise de dados e previsão de resultados baseados em campanhas de publicidade em redes sociais.

Cada pasta geralmente contém seus próprios conjuntos de dados (como arquivos `.csv`), scripts em Python (`.py`) e Jupyter Notebooks (`.ipynb`) detalhando o passo a passo das análises exploratórias, processamento de dados e treinamento dos modelos.

## 🚀 Como Executar

Para rodar os projetos localmente na sua máquina, siga os passos abaixo:

1. **Pré-requisitos**: Certifique-se de ter o [Python](https://www.python.org/) instalado em sua máquina.
2. **Clone o repositório**:
   ```bash
   git clone https://github.com/saviosoaresUFC/Learning-AI.git
   cd Learning-AI
   ```
3. **Ambiente Virtual (Recomendado)**:
   Crie e ative um ambiente virtual para instalar as dependências de forma isolada.
   ```bash
   python -m venv venv
   # No Windows
   venv\Scripts\activate
   # No Linux/Mac
   source venv/bin/activate
   ```
4. **Instale as dependências**:
   As bibliotecas utilizadas podem variar entre os módulos, mas em geral, você precisará dos pacotes clássicos de ciência de dados:
   ```bash
   pip install pandas numpy scikit-learn matplotlib seaborn jupyter
   ```

## 🧩 Visualização de Notebooks (Extensão Colab / Jupyter)

Este projeto faz uso extensivo de **Jupyter Notebooks** (arquivos `.ipynb`) para documentar e executar os códigos em blocos iterativos.

⚠️ **IMPORTANTE**: Se você estiver utilizando o **Visual Studio Code (VS Code)**, para uma melhor experiência, formatação e visualização correta desses arquivos, **é estritamente necessária a instalação das extensões corretas**:

- **Extensão Jupyter/Colab**: Instale a extensão oficial do `Jupyter` ou extensões baseadas no ambiente do `Google Colab` no seu VS Code. Isso permitirá que você abra os arquivos `.ipynb`, visualize os gráficos gerados, leia a documentação em Markdown renderizada corretamente e execute as células de código passo a passo.

*Alternativamente*, se não quiser configurar o ambiente local, você pode fazer o upload dos arquivos `.ipynb` e seus respectivos `.csv` diretamente para o ambiente na nuvem do [Google Colab](https://colab.research.google.com/) e executá-los pelo navegador de forma rápida e gratuita!
